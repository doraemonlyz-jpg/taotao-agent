ℹ️ About

关于这套手册

这套 24 本手册是为 2026 年 senior agent engineer 面试系统化整理的——不是教程合集 · 是按 "从 0 到能上岗" 路径设计的完整体系。2026-05 新增 4 本 senior 必补章节（行为面 / Post-Training / Multimodal & Voice / Agent Benchmarks）+ 3 大补丁（LLM 30 题速查 / Cost-Latency 5 case / Security 30 招攻防）+ Book 22 工业化基建（反编译 Claude Code 17 features）+ Book 23 Multi-Agent Architecture（5 模式 + 4 框架横评 + "默认别用" 判断框架）+ Book 24 From Demo to Public SaaS（11 phase 产品化路线图 + 50 项上线 checklist + 配套 auth / 多租户 / nginx / Dockerfile.prod 4 starter impl）。

📐 设计原则

  1. 面试导向——每本配 12-50 道 senior 面试题 + 答题模板 · 不是 "了解一下" 而是 "能上场答"
  2. 覆盖完整——21 本各有独立主题 · 横跨技术 (Books 01–17) + 软实力 (Book 18) + 高阶专题 (Books 19–21)，覆盖 Anthropic / OpenAI / Stripe / Cursor / Mistral / 字节 等头部 senior 流程的每一轮
  3. 真坑 vs 玄学——每本都有 "翻车复盘" / "面试三档答案" · 别人用真血换的经验
  4. 引用一手资料——官方文档 / 源码 / paper · 不抄博客抄博客
  5. 跟 demo 同步——本项目 Book 02 / 13 / 15 / 16 / 17 / 18 都有真实可跑的代码对照 · 改了代码就改书
