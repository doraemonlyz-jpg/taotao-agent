ℹ️ About

关于这套手册

这套 17 本手册是为 2026 年 senior agent engineer 面试系统化整理的——不是教程合集 · 是按 "从 0 到能上岗" 路径设计的完整体系。

📐 设计原则

  1. 面试导向——每本配 12-50 道 senior 面试题 + 答题模板 · 不是 "了解一下" 而是 "能上场答"
  2. 互不重复——17 本各有独立主题 · 但内部 cross-link 把相关概念串起
  3. 真坑 vs 玄学——每本都有 "翻车复盘" / "面试三档答案" · 别人用真血换的经验
  4. 引用一手资料——官方文档 / 源码 / paper · 不抄博客抄博客
  5. 跟 demo 同步——本项目 Book 02 / 13 / 15 / 16 / 17 都有真实可跑的代码对照 · 改了代码就改书

🛠 配套项目

github.com/doraemonlyz-jpg/taotao-agent · 完整生产级 agent demo · 同时支持 graph 和 harness 两种实现（POST /chat + POST /chat/v2）· LangGraph + supervisor + sub-agent + HITL + memory + tool + eval + MCP 双向（暴露/消费 tools）+ P1 加固（cost 护栏 / memory decay / parallel exec / OpenAPI codegen / multi-session UI）· 跟 Book 02 (graph 实现) · Book 13 (harness 实现) · Book 16 (MCP) · Book 17 (P1) 配套读 · 直接 clone 跑得起。
