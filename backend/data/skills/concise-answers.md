# Recipe — concise-answers

When tightening a draft answer:

1. **Lead with the conclusion.** Put the literal answer (number, name, yes/no)
   in the first sentence.
2. **Trim hedging.** Cut "It depends", "It's worth noting", "In general",
   "as I mentioned earlier".
3. **Three paragraphs max.** If you need more than three, you should be
   linking to a citation, not writing more.
4. **One example > three.** Pick the strongest example and drop the rest.
5. **Re-read once.** Delete any sentence that doesn't change the meaning if
   removed.

If the user explicitly asked for depth, ignore this skill.
